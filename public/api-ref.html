Base URL: https://maids.ng/api  |  Authenticated calls require Authorization: Bearer {token}

Authentication

POST/v1/auth/registerpublic

Register a new user (maid or employer). Returns a Bearer token.

Example Queries

User Login Flow

# Login as employer
curl -s -X POST https://maids.ng/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"jane@example.com","password":"secret123"}' | jq

# Extract token
TOKEN=$(curl -s -X POST https://maids.ng/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"jane@example.com","password":"secret123"}' | jq -r '.data.token')

# Get profile
curl -s https://maids.ng/api/v1/auth/me \
  -H "Authorization: Bearer $TOKEN" | jq

# Search maids
curl -s "https://maids.ng/api/v1/maids?location=Lagos" \
  -H "Authorization: Bearer $TOKEN" | jq

Admin Queries via MCP Token

MCP_TOKEN="your-mcp-token"
BASE="https://maids.ng/api"

# List all active maids
curl "$BASE/v1/cli/maids?status=active" -H "Authorization: Bearer $MCP_TOKEN" | jq

# Get maid detail with verification status
curl "$BASE/v1/cli/maids/13" -H "Authorization: Bearer $MCP_TOKEN" | jq

# List all employers
curl "$BASE/v1/cli/users?role=employer" -H "Authorization: Bearer $MCP_TOKEN" | jq

# User summary for decision-making
curl "$BASE/v1/cli/users/51/summary" -H "Authorization: Bearer $MCP_TOKEN" | jq

# Pending matching jobs
curl "$BASE/v1/cli/matching/queue?status=pending" -H "Authorization: Bearer $MCP_TOKEN" | jq

# Active assignments
curl "$BASE/v1/cli/assignments?status=active" -H "Authorization: Bearer $MCP_TOKEN" | jq

# Wallet balance
curl "$BASE/v1/cli/wallet?user_id=51" -H "Authorization: Bearer $MCP_TOKEN" | jq
