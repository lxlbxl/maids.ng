# Maids.ng

Welcome to the Maids.ng platform!

## Description
This repository contains the source code for the Maids.ng platform. It includes the frontend interface and the PHP backend API designed to match maids with employers seamlessly.

## Project Structure
- `backend/` - The backend API powered by PHP.
- `*.html` - Frontend HTML pages for the user interface.
- `js/` - Frontend JavaScript files.
- `images/` - Static images and icons used on the site.

## Setup Instructions
1. Clone the repository.
2. Ensure you have PHP set up (e.g. XAMPP, Laragon, or custom Nginx/Apache setup) pointing to this directory.
3. Check the `backend/` directory for any additional `.env` or configurations needed.
4. Open `index.html` to start exploring the frontend.
